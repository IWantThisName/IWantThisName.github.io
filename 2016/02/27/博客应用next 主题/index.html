博客应用next主题

发表于 2016-02-27 | 分类于 hexo | | 热度

下载next主题并应用到hexo博客上

  1. 下载next主题
    点击这里按照下图进行下载

  2. 下载完成后进行解压

    将解压后的文件夹名字重命名为next,放到blog目录下的theme文件夹下

  3. 修改blog目录下的_config.yml站点配置文件

  4. 在blog目录下按住shift+右键,打开命令行窗口

  5. 在命令行里键入以下命令

    D:\nodejs\blog> hexo g
D:\nodejs\blog> hexo d

    然后重新键入你的博客页面,就可以看到页面已经更换了

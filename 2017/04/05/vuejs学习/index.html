Vue.js的Hello World

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
    <!--这是一个view-->
    <div id='app'>
        {{ message }}
    </div>
</body>
<script src="vue.js"></script>
<script>
    <!--这是一个Model-->
    var exampleData = {
        message:'Hello World!'
    }

    //一个vue实例 连接model和view
    new Vue({
        el:'#app',
        data:exampleData
    })
</script>
</html>

vue.js是一个MVVM模式的js库,修改数据相应的修改样式 = = 只能理解这么多了

Vue.js的一些常用指令

vue的常用指令是用来进行一些条件渲染的

v-model

绑定相应数据到标签

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
    <div id="app">
    <p>{{ message }}</p>
    <!-- 将message的值绑定到input标签上  input标签上的值也绑定到message上 
    修改input的值就修改message的值 -->
    <input type="text" v-model="message"/>
    </div>
</body>
<script src="vue.js"></script>
<script>
    var exampleData = {
        message:'Hello World'
    }

    new Vue({
        el:'#app',
        data:exampleData
    })
</script>
</html>

v-if

根据后面的条件表达式或者布尔值决定生成不生成相应的元素 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
    <div id='app'>
        <h2>Hello,Vue.js</h2>
        <h3 v-if="yes">Yes</h3>
        <h3 v-if="no">No</h3>
        <h3 v-if="age >= 25">Age:{{ age }}</h3>
        <h3 v-if="name.indexOf('jack') >= 0">Name:{{ name }}</h3>
    </div>
</body>
<script src="vue.js"></script>
<script>
    new Vue({
        el:"#app",
        data:{
            yes:true,
            no:false,
            age:28,
            name:'Goodson'
        }
    })
</script>
</html>

v-show

根据后面的条件表达式或者布尔值具体元素显示不显示相应的元素,与v-if的区别就是v-show是将全部 元素都生成到页面的,只是当条件表达式结果或者布尔值为false时,设置元素的style属性的display属性值为none,不显示该属性,但是该元素在html页面中还是存在的 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
    <div id='app'>
        <h2>Hello,Vue.js</h2>
        <h3 v-show="yes">Yes</h3>
        <h3 v-show="no">No</h3>
        <h3 v-show="age >= 25">Age:{{ age }}</h3>
        <h3 v-show="name.indexOf('jack') >= 0">Name:{{ name }}</h3>
    </div>
</body>
<script src="vue.js"></script>
<script>
    new Vue({
        el:"#app",
        data:{
            yes:true,
            no:false,
            age:28,
            name:'Goodson'
        }
    })
</script>
</html>

v-else

v-else得和v-if或者v-show一起用,当v-if或者v-show的表达式结果为false时,v-else的内容就会显示(不知道啥毛病 我的v-show后面跟的v-else不起作用,只能跟v-if一起用)

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
    <div id="app">
        <h1>Hello Vue.js</h1>
        <h2 v-if="yes">Yes</h2>
        <h2 v-else>No</h2>
        <h2 v-show="age >= 25">Age:{{ age }}</h2>
        <h2 v-else>年龄太小</h2>
    </div>
</body>
<script src="vue.js"></script>
<script>
    new Vue({
        el:"#app",
        data:{
            yes:false,
            age:23
        }
    })
</script>
</html>

v-for

v-for指令基于一个数组渲染一个列表，它和JavaScript的遍历语法相似：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<div id="app">
    <table>
        <tr>
            <th>name</th>
            <th>age</th>
            <th>sex</th>
        </tr>
        <tr v-for="person in persons">
            <td>{{ person.name }}</td>
            <td>{{ person.age }}</td>
            <td>{{ person.sex }}</td>
        </tr>
    </table>
</div>
</body>
<script src="vue.js"></script>
<script>
    new Vue({
        el:"#app",
        data:{
            persons:[
                {
                    name:"aaa",
                    age:21,
                    sex:'男'
                },
                {
                    name:"bbb",
                    age:22,
                    sex:'女'
                },
                {
                    name:"ccc",
                    age:24,
                    sex:'男'
                }
            ]
        }
    })
</script>
</html>

v-bind

v-bind指令可以在其名称后面带一个参数，中间放一个冒号隔开，这个参数通常是HTML元素的特性（attribute），例如：v-bind:class

<body>
    <div id="app">
        <ul class="pagination">

            </li>

<script src="vue.js"></script>
<script>
    new Vue({

</html>
